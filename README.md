# Women's Beauty Salon Website

Multi-page, responsive salon website with a luxury feminine design system.

## Pages

- `index.html` - Home
- `about.html` - About Us
- `services.html` - Services
- `pricing.html` - Pricing
- `gallery.html` - Before & After Gallery
- `booking.html` - Online Booking
- `contact.html` - Contact + Map

## Features

- Soft pastel/nude palette with gold accents
- Smooth scroll reveal animations and hover effects
- Mobile-first responsive navigation
- Booking form with browser-side appointment capture (`localStorage`)
- Floating WhatsApp CTA and call links
- SEO-friendly meta titles/descriptions
